# Pilot Playbook — Grounded LLM

**Пакет:** Policy Assistant Pilot (8 недель)  
**Покупатель:** спонсор из HR / IT, компания 200–5 000 сотрудников

---

## Что входит

| Пункт | Детали |
|-------|--------|
| Область | 1 домен знаний (например HR policies) |
| Документы | До 50 файлов или 200 МБ |
| Пользователи | До 500 активных |
| Деплой | On-prem / private cloud (Docker) |
| Каналы | Web-чат (+ Telegram опционально) |
| Язык | English или **русский** (`DEFAULT_LOCALE=ru`) — см. [SALES_ONE_PAGER_RF.md](./SALES_ONE_PAGER_RF.md) |
| Поддержка | 2 созвона в неделю на пилоте |

**Ориентир цены пилота:** $8 000 – $25 000 USD.

---

## Таймлайн (8 недель)

| Неделя | Работы |
|--------|--------|
| **1** | Kickoff, security questionnaire, список документов |
| **2** | Деплой, сеть, LLM API |
| **3** | Загрузка документов, reindex, промпты |
| **4** | UAT с HR (10–20 человек) |
| **5** | Soft launch 50–100 сотрудников |
| **6** | Масштаб до 200–500, feedback |
| **7** | Доработка провалов, новые документы |
| **8** | Отчёт + go/no-go на годовую лицензию |

**Demo-ready:** конец 3-й недели.

---

## KPI успеха

| Метрика | Цель |
|---------|------|
| In-scope ответы с цитатами | ≥85% |
| Verify pass rate | ≥75% |
| 👍 на сообщения ассистента | ≥70% |
| P95 время ответа | <15 с |

«Нет в базе» — **нормальный** ответ, не ошибка.

## Demo 30 минут

1. **Проблема (3 мин)** — повторяющиеся вопросы в HR; ChatGPT нельзя.
2. **Чат (10 мин)** — [DEMO_SCRIPT.md](./domain-packs/DEMO_SCRIPT.md).
3. **Доверие (5 мин)** — цитаты, verify, on-prem.
4. **Админка (5 мин)** — upload → reindex.
5. **Enterprise (5 мин)** — API, OpenAPI, roadmap.
6. **Закрытие (2 мин)** — пилот 8 недель, лицензия.
