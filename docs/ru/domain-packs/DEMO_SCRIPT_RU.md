# Сценарий HR-demo для рынка РФ (30 минут)

Домен **default**, локаль **ru** (`DEFAULT_LOCALE=ru` или `?locale=ru`).

Демо-документы: `data/default/policy_*.txt` (без суффикса `_en`).

---

## Подготовка

```bash
cp .env.example .env
```

Минимум в `.env` для demo:

```env
DEFAULT_LOCALE=ru
TELEGRAM_AUTH_DISABLED=true
ADMIN_PASSWORD=admin
ADMIN_SECRET=local-dev-secret
LLM_API_KEY=<ваш ключ>
LLM_BASE_URL=<endpoint>
LLM_MODEL=<модель>
```

```bash
docker compose up -d --build
python scripts/reindex_rag.py
```

Проверка retrieval (12 кейсов, без LLM):

```bash
pip install requests
python scripts/run_rag_eval.py --suite default
```

**URL:**

- Чат: `http://localhost/?locale=ru`
- Админка: `http://localhost/admin.html`

> `TELEGRAM_AUTH_DISABLED=true` — **только для demo в браузере**. В production у клиента — Telegram Mini App или API keys.

## Вступление (3 мин)

> «Сотрудники каждую неделю задают HR одни и те же вопросы: отпуск, больничный, удалёнка. Публичный ChatGPT для внутренних политик нельзя — политика ИБ и 152-ФЗ.  
> Grounded LLM отвечает **только из ваших документов**, с **цитатами**, разворачивается **в вашей инфраструктуре**. Если ответа нет в базе — говорит честно, без галлюцинаций.»

## Live-вопросы (10 мин)

Задавайте в web-чате. На каждый ответ показывайте **текст + блок «Источники»**.

## Доверие (5 мин)

1. **Цитата** — покажите имя файла (`policy_vacation.txt` и т.д.) в блоке источников.
2. **Verify** — объясните: система проверяет цифры и факты; выдуманные числа блокируются или смягчаются.
3. **Данные** — документы, Postgres, Chroma — у клиента; embeddings локально; LLM — endpoint на выбор клиента.
4. **Не обучаем** — документы клиента не идут на fine-tuning публичных моделей.

---

## Админка (5 мин)

1. Открыть `/admin.html`, войти (Basic Auth).
2. Загрузить небольшой `.txt` с новым правилом (например: «Командировки согласуются за 7 дней»).
3. Нажать **Reindex RAG**.
4. Задать в чате вопрос, на который отвечает только новый файл.

> Показывает: HR может обновлять базу без разработчиков.

## Troubleshooting

| Проблема | Проверка |
|----------|----------|
| Пустые ответы | `docker compose logs python`; выполнить reindex |
| 401 в браузере | `TELEGRAM_AUTH_DISABLED=true` в `.env`, перезапуск server |
| UI на английском | `DEFAULT_LOCALE=ru` или `?locale=ru` |
| Eval падает | Chroma запущен; `python scripts/reindex_rag.py` |
| Медленные ответы | LLM endpoint / локальная модель; проверить `LLM_BASE_URL` |
